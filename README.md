# SOLAR-POWER-BASED-MULTIPURPOSE-AGRI-ROBOT-USING-ANDRIOD-APPLICATION-
This project develops a multitasking agricultural robot to reduce labor and improve efficiency. Using Arduino UNO and HC-05 Bluetooth, it performs ploughing, seed dispensing, and water sprinkling. It supports smart farming with low cost, eco-friendly energy, and automation for better productivity.

# Solar Power Based Agriculture Robot 🌱☀️

## 📌 Description
This project is designed to help farmers by using solar energy. It automates irrigation and reduces manual work.

## 🚀 Features
- Solar powered system
- Automatic irrigation
- Low cost and eco-friendly

## 🛠️ Technologies Used
- Arduino
- Soil moisture sensor
- Solar panel
- Motor driver

## ⚙️ Working
The system uses solar energy to power the robot. It checks soil moisture and automatically waters plants when needed.
